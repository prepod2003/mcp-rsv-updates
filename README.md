# MCP:RSV Server — канал обновлений

Это P2-репозиторий обновлений плагина **MCP:RSV Server** для 1C:EDT.

Плагин обновляется автоматически — вручную сюда ходить не нужно. Если хотите
установить обновление вручную, в 1C:EDT откройте
**Help → Install New Software…** и укажите адрес:

```text
https://prepod2003.github.io/mcp-rsv-updates/p2/standard/
```
